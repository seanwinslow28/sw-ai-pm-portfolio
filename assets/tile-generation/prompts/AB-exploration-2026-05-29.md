# Tile A/B Exploration Prompts — 2026-05-29

For AI Studio (Nano Banana 2 / Gemini) + ChatGPT (GPT-image). Generate, compare, pick.

**How to use**
- Set aspect ratio **4:5 portrait** (tiles are 400×500, cover-cropped). Target ~1200×1500.
- Every prompt is self-contained and **text-free by design** — type (title, anchor metric) is added later as an HTML overlay, never baked into the art.
- Palette anchors: warm paper `#FFF9F0`, ink/graphite near-black, teal `#0A3E42`, one restrained amber accent.
- **Path A** = one cohesive pencil-test register across all tiles. **Path B** = each tile in its own medium (the mynrd gallery model).
- Two attempts per prompt, then move on. Commit the winners' prompts back here.
