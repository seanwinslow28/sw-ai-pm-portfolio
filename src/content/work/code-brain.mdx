<Opener>
  <p>At 08:45 every weekday, a launchd job on a Mac Mini in my apartment wakes up a Claude agent named Daily Driver. It reads last night's git commits, scans the vault for anything I touched, opens four JSON files in the portfolio repo, and writes the day's dateline into them before the coffee finishes brewing. I used to call this folder my second brain. The folder did not change. The folder has a fleet now.</p>
  <p>The vault was a Markdown graveyard for two years. Meeting notes, half-written essays, a 4,000-word file called "thoughts-on-pacing.md" that I have never re-opened. Then the Claude Agent SDK shipped, and the same folder started hosting things that ran on a clock. 118 skills now compose into 8 named agents the way recipes compose into a kitchen. The Daily Driver writes the morning brief. Vault Synth runs against new commits at 14:00. Process Inbox triages overnight email before I've made eye contact with a single human. None of them are mine in any honest sense. I named them, scheduled them, and they out-produced me by Tuesday of week one.</p>
  <p>The fleet replaced the guy who organized media files in a basement for nine years. Same hands, different tools. He's still in the basement. The basement just has a Mac Mini in it now. What's below is the morning ritual. The bill isn't in dollars.</p>
</Opener>
