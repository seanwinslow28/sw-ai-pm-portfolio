## Voice calibration notes (carry forward from prior sessions)

These are non-negotiable. Read the full ceiling rules in §"Carry-forward constraints" of `case-study-openers-locked-2026-05-26.md`. Quick reference:

1. **Quick and silly, NOT heavy self-deprecation.** Forward-tilted, not retrospective-confessional.

3. **No fixed counts on Code Brain.** Skills/agents/hooks fluctuate weekly. Use "a fleet that grows and shrinks with the work" or omit the count.

6. **"Building got cheap" ceiling reached.** About B-2 ¶2 + B-4 ¶1. **Do not use in taglines.**
7. **HybridRouter STOP-DOING.** Methods strip cells only, never in user-facing prose.
8. **Watch council-default reflexes.** When multiple drafts reach for the same anchor unprompted (e.g., "cold coffee," "5 a.m.," timestamps that smell like grinding), it's the safe call, not the strongest call. Push past it.
9. **Repetition discipline.** Accidental over-use of small words = Bad Sean. Intentional refrains/parallels = fine.
10. **Wire-service-style format, sentence-case prose.** Taglines can be all-sentence-case (no lowercase-only aesthetic).
11. **Compound nouns/hyphens for parallel comparisons.** Fuse compared options into symmetric units.
12. **Specific nouns over abstract claims.** Always anchor.
13. **Kill the "I realized that…" / "I used to think…" / "I noticed…" signposts.** Pivots happen by collision.
14. **Page-level callback rhymes are gold** — but watch the ceilings above.
